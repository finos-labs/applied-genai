# FINOS Applied GenAI Community Events

FINOS conducts hackathons to rapidly prototype Generative AI solutions for financial services challenges. These intensive events bring together practitioners from major financial institutions, technology providers, and academic institutions to collaborate on real-world problems.

### Hackathon Objectives

| Objective | Description |
|-----------|-------------|
| **Rapid Innovation** | Build working prototypes in 24-48 hours |
| **Cross-Functional Teams** | Connect developers, business analysts, and domain experts |
| **Knowledge Sharing** | Learn from peers across the industry |
| **Open Source** | Create reusable tools for the community |
| **Community Building** | Strengthen financial services open-source ecosystem |

## Structured Finance Hackathons

**Focus Areas:**
Leveraging AI and ML technologies to address critical challenges in structured finance, disaster risk modeling, agent workflows, smart transaction routing, and linking fragmented systems.

**Key Challenges:**
- **Agent Workflows**: Building intelligent automation for financial processes
- **Disaster Risk Modeling**: Climate and weather risk assessment for portfolios
- **Smart Transaction Routing**: Optimizing payment and settlement flows
- **System Integration**: Connecting fragmented legacy infrastructure
- **Invoice Financing**: Improving access to capital for SMEs

**Focus Areas:**
- Trade processing automation and exception handling

- Data quality validation
- Operational efficiency optimization

**Key Challenges:**
1. **Automated Trade Exception Resolution**: Use GenAI to identify, categorize, and resolve trade breaks
2. **Intelligent Regulatory Reporting**: LLM-powered report generation and validation
3. **Risk Data Aggregation**: AI-driven data normalization and analysis

**Technologies Used:** Python, LLaMA, GPT, RAG, LangChain, CDM
